# Business Sales Analysis Dashboard

## Objective

Analyze business sales data to identify revenue trends, top-selling products, high-value categories, and regional performance.

---

## Tools Used

- Power BI
- Microsoft Excel
- GitHub

---

## Dataset

Business Sales Dataset

---

## Dashboard Features

- KPI Cards
- Revenue Trend Analysis
- Top-Selling Products
- Category-wise Sales
- Regional Performance
- Sales Distribution
- Interactive Slicers

---

## Key Insights

- Identified top-performing products based on sales.
- Compared revenue across different product categories.
- Analyzed regional sales performance.
- Identified monthly sales trends.
- Highlighted high-value categories contributing most to revenue.

---

## Recommendations

- Focus marketing efforts on high-performing products.
- Increase inventory for top-selling categories.
- Improve sales strategies in low-performing regions.
- Monitor monthly sales trends for better forecasting.
